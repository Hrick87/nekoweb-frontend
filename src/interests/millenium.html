MILLENIUM

What is Millenium?

Millenium is a TTRPG focused on survival in an apocolyptic gonzo-fantasy world, weapon building, monster hunting, and strategic momentum shooter based combat.

Features

  • Classless Character Creation
  • Easy to learn 2d10 rolls for everything
  • In-depth weapon customization
  • Spontaneously modifiable "magic system"
  • Simple but intricate crafting system
  • A simple but robust influence system
  • Fauna and plant life have actual system weight
  • Unique combat system that uses typical movement shooter gimmicks like wall running and knockback

How can I play?

As of now the game is in a private testing stage. After this has finished, a public testing phase will be announced and linked here.
